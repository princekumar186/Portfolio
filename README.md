# 🌐 Personal Portfolio Website

A modern and responsive **personal portfolio website** built to showcase my **projects, skills, certifications, and contact information**.
The website acts as a digital profile where visitors can explore my work, view project demonstrations, and connect with me.

This portfolio highlights my development skills in **web development, programming, and software projects**.

---

## 🚀 Features

* Modern and responsive design
* Project showcase section
* Image slider for project previews
* Skills and technologies section
* Certifications gallery
* Contact section with email integration
* Smooth animations and interactive UI
* Mobile-friendly layout

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Responsive Web Design**
